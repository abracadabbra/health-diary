# 🌿 养生日记

记录每日健康生活，养成良好生活习惯。

## 📁 项目结构

```
health-diary/
├── README.md          # 项目说明
├── diary/             # 每日日记
│   └── 2026/          # 按年份组织
│       └── 05/        # 按月份组织
├── templates/         # 日记模板
└── docs/              # 养生知识
```

## 📝 日记格式

每天记录以下内容：

- **日期**：YYYY-MM-DD
- **天气**：晴/阴/雨/雪
- **睡眠**：时间、质量
- **饮食**：三餐记录
- **运动**：类型、时长
- **心情**：1-10分
- **养生小记**：今日养生心得
